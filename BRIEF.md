# Portfolio Redesign — Project Brief

## What this is
A ground-up redesign of allenchong.studio, moving away from Carrd
toward a custom-hosted site built in Next.js (App Router, TypeScript,
CSS Modules). Domain already owned. Figma design is complete for all
core pages. Code is in progress — Home, Nav, and CS01 are built and
running locally.

## Owner
Allen Chong — UX/content designer and strategist, Senior Staff level 
(formerly Meta). Open to full-time and contract work; preference for 
well-paid discrete consulting engagements, but actively pursuing both 
given current market.

## Target audience
- Hiring managers and recruiters at tech companies, agencies, or 
  consultancies
- Both full-time and contract/consulting clients
- Lead with the work; let the About page carry availability nuance

## The narrative
Allen's career has a clear throughline that most content designers 
lack: every role — teacher, assessment designer, curriculum designer, 
UX/content designer — has been about taking complex ideas and making 
them understandable so people can act with confidence and autonomy. 
That's not a winding career path. It's a compounding one.

## Desired impression
Visitors should leave feeling that Allen is a uniquely skilled content 
designer and strategist whose breadth (education, assessment, 
curriculum, UX, marketing) is not scattered but coherent — all in 
service of the same core ability: clarity that empowers.

## Tone
Editorial. Confident without being boastful. Warm but not casual. 
Precise — no filler phrases, no AI-sounding constructions.

## Pages
- Home
- Case Studies (Messenger Communities — 3 studies)
- Highlights (formerly Selected Work)
- About

## Tech stack (finalized)
- Next.js 16.2.2 (App Router, static generation, TypeScript)
- CSS Modules with design tokens in globals.css
- Single 960px max-width container, 48px side padding (24px mobile)
- Responsive breakpoints: 1056px, 820px, 720px
- Figma file key: aAOF7I0rz5hPSQJ724nNVc (v2 page, node 105:199)

## Out of scope for v1
- Dark mode
- Color accents beyond terracotta
- Font weight variation in display type
- Deployment/hosting setup (future phase — Allen will need guidance)
